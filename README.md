# badges

A collection of GitHub achievement badges and how to earn them.

## Badges Overview

| Badge | How to Earn |
|-------|-------------|
| ⚡ Quickdraw | Close an issue or PR within 5 minutes of opening |
| 🦈 Pull Shark | Open pull requests that get merged (2/16/128/1024) |
| 😅 YOLO | Merge a PR without code review |
| 🤝 Pair Extraordinaire | Co-author commits on merged PRs |
| 🧠 Galaxy Brain | Have replies accepted as answers in Discussions |
| ⭐ Starstruck | Have a repository receive 16+ stars |
| ❤️ Public Sponsor | Sponsor a contributor via GitHub Sponsors |
